  <h2 id="faq" style="margin-top:30px;">Frequently asked questions (CSV to XML)</h2>

  <h3 id="faq-1">Does the tool use CSV headers as XML tags?</h3>
  <p>Yes. The first row of your CSV automatically becomes the names of the child elements (tags) inside each row node. For example, headers like “Name” and “Price” become <code>&lt;Name&gt;</code> and <code>&lt;Price&gt;</code>.</p>

  <h3 id="faq-2">What XML structure does it generate?</h3>
  <p>The output is a hierarchical XML document. Each CSV row becomes a <code>&lt;row&gt;</code> element (or similar), containing child elements for each column. Special characters are escaped, and the output is indented for readability.</p>

  <h3 id="faq-3">Does it handle special characters like &lt;, &gt;, and &amp;?</h3>
  <p>Yes. All XML‑sensitive characters are properly escaped: <code>&lt;</code> becomes <code>&amp;lt;</code>, <code>&amp;</code> becomes <code>&amp;amp;</code>, etc., ensuring valid XML output.</p>

  <h3 id="faq-4">Can I edit the CSV data before converting to XML?</h3>
  <p>Absolutely. The CSV preview table is fully editable. You can change values, add or remove rows, then generate XML – all live in your browser.</p>

  <h3 id="faq-5">Is my data sent to a server?</h3>
  <p><strong>No.</strong> The tool runs entirely in your browser using the SheetJS library. Your file never leaves your computer – that’s why it works offline too.</p>
